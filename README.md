# 🚦 Smart Traffic Light System (ESP32)

## 📌 Description
This project is a smart traffic light system built using ESP32.  
It simulates traffic control for a 4-road intersection and can be extended with sensors to optimize traffic flow.

## ⚙️ Features
- Controls traffic lights (Red, Yellow, Green)
- Simulates a 4-way intersection
- Can be integrated with sensors for smart decision making

## 🧰 Components Used
- ESP32
- LEDs (Red, Yellow, Green)
- Resistors
- Breadboard
- Jumper wires

## 💻 Simulation
This project was tested using Wokwi simulator.

## ▶️ How to Run
1. Open the project in Wokwi
2. Upload the code to ESP32
3. Run the simulation
